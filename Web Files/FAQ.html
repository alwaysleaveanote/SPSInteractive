FAQs

  • How can I create my own set of schools to compare?

    You can select specific schools in two ways. First, you can ctrl + click each of the schools you are interested in, and the charts will filter accordingly. Alternatively, you can hover over the map, hover over the bottom option on the map toolbar (the one that looks like a play button), and select one of the three rightmost options. This will allow you to draw an area on the map, and select all the schools within that area. The charts will then filter accordingly.
    Unfortunately, there is no way to save this selection once it is made, so you will have to reselect the schools each time you visit a new page.

  • I moved the map! How do I reset it?

    To recenter and resize the map, so that it shows all the options, simply hover over the map, and click on the "home" option on the toolbar.

  • Can I download the data you used?

    All the data used comes from the OSPI's website. You may either download it from there, or go to the chart you wish to get the data from, click on the left axis and hover over it, and choose the "view data" option. From there, you can download all the data used to create the chart.

  • Can I save a chart I created?

    Unfortunately, it is not currently possible to save any custom charts you create. Write down the combination of charts and maps you used, so you can recreate it!
